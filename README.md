# Guessing Number Game

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/guessing-number-game.git
```

### Step 2

Change the directory to guessing-number-game

```shell
cd guessing-number-game
```

### Step 3

Run project

```shell
python main.py
```
